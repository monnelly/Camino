# 🥾 Camino de Santiago · Pilgrim's Journal

A daily journal app for pilgrims walking the Camino de Santiago (Camino Francés).

## Features

- **Daily checklist** — reflection, conversation, selfie, standout photo
- **Photo storage** — upload and store your selfie and photo of the day
- **Written notes** — reflection and conversation journals per day
- **Geographic map** — real Leaflet map tracing your route across Spain
- **Progress tracker** — km walked, % to Santiago, days remaining
- **Gallery** — all your memories in one scrollable view

## Live App

Deployed at: `https://YOUR_USERNAME.github.io/YOUR_REPO_NAME`

## Local Development

```bash
npm install
npm run dev
```

## Deploy

This app auto-deploys to GitHub Pages via GitHub Actions on every push to `main`.
